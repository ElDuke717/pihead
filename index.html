Metrics Explanation

  • Disk R/W Rates: The rate at which data is being read from or written to the disk. If you see "iostat: not found", it indicates the required tool isn't installed on the Raspberry Pi.
  • Active Connections: The number of active network connections to the Raspberry Pi.
  • CPU Frequency: The current frequency at which the CPU is running. Typically measured in Hz.
  • CPU Usage: The percentage of CPU currently being used by the system.
  • Disk Space: The percentage of disk space currently being used.
  • GPU Memory: Amount of memory currently being used by the GPU (Graphics Processing Unit).
  • Load Average: Average system load over 1, 5, and 15 minutes intervals. A value of 1.0 means the CPU is fully utilized.
  • Memory Usage: The percentage of total memory currently being used.
  • Network Usage: These numbers represent the rate of data being sent and received over the network. They might typically be measured in units like Megabits per second (Mbps) or Kilobits per second (Kbps). Here's a breakdown:
    • 1st Value (0.00): Average data rate received over the last minute.
    • 2nd Value (0.00): Average data rate sent over the last minute.
    • 3rd Value (0.21): Average data rate received over the last 5 minutes.
    • 4th Value (0.34): Average data rate sent over the last 5 minutes.
  • Running Processes: The number of processes currently running on the Raspberry Pi.
  • Temperature: Current temperature of the Raspberry Pi's CPU in degrees Celsius.
  • Uptime: The duration for which the Raspberry Pi has been running continuously.
